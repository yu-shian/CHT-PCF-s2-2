# ISO 14067 Product Carbon Footprint Calculator

A React-based calculator for ISO 14067 product carbon footprint (PCF) assessment.

## Getting Started

### Prerequisites
- Node.js (v20 or newer recommended)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

### Build

Build for production:
```bash
npm run build
```

## Deployment

This project is configured to deploy to **GitHub Pages** automatically via GitHub Actions.

1. Go to your repository **Settings** -> **Pages**.
2. Under "Build and deployment", select **GitHub Actions** as the source.
3. Push changes to the `main` branch.
4. The site will be built and deployed automatically.
